Familiar Languages

Java

Java is the main language that I'm familiar with. Training that I've recieved with Java includes a week long introduction to Java at ID Tech Camps, a Semester 1 introduction to Java (CSC-120) course at the University of Miami which covered basic Java coding techniques, and a Semester 2 programming course (CSC-220) covering basic data structures, sorting, and manipulating data. Javascript

Javascript is the language I have the most fun with. Almost all of my pet projects are written in javascript with the help of the p5.js library. This library allows its users to create interactive graphical content and can be used for making games on basic HTML driven pages. I learned all of my Javascript knowledge through my Semester 1 Creative Coding course (CIM-540). In this course, I learned how to utilize the p5.js library to create multiple games and graphics. To see some of these projects, check out my projects page.
